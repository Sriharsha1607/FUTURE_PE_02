# FUTURE_PE_02 – AI Content Marketing using UGC Ads

## Project Overview

This project demonstrates how Prompt Engineering can be used to generate high-conversion UGC (User Generated Content) ad scripts for short-form platforms like Instagram Reels and YouTube Shorts.

## Product

Flowlog – Productivity SaaS Tool

A single workspace for freelancers and solo founders to manage tasks, notes, and time without switching between multiple apps.

## Problem

Freelancers and solo founders lose 2–3 hours daily switching between tools like Notion, Trello, and calendar apps.

## Solution

Flowlog combines all essential tools into one focused workspace, reducing distraction and improving productivity.

## Target Audience

• Freelancers
• Solo founders
• Indie hackers
• Age: 24–38

## Objective

Design a reusable prompt system that generates:

• UGC ad scripts
• Scroll-stopping hooks
• Call-to-action (CTA) lines
• Platform-specific adaptations

## Platforms

Instagram Reels
YouTube Shorts

## Tools Used

# FUTURE_PE_02 – AI Content Marketing using UGC Ads

## Project Overview

This project demonstrates how Prompt Engineering can be used to generate high-conversion User Generated Content (UGC) ad scripts for short-form video platforms like Instagram Reels and YouTube Shorts.

Unlike traditional advertising, UGC ads feel natural, relatable, and authentic — making them significantly more effective in capturing attention and driving engagement.

---

## Product

Flowlog – Productivity SaaS Tool

A single workspace designed for freelancers and solo founders to manage tasks, notes, and time without switching between multiple applications.

---

## Problem Statement

Freelancers and solo founders often use multiple tools like Notion, Trello, and calendar apps, leading to:

- Constant context switching
- Loss of focus
- Wasted 2–3 hours daily
- Reduced productivity

---

## Solution

Flowlog solves this problem by combining all essential productivity tools into one focused workspace, helping users save time and stay organized.

---

## Target Audience

- Freelancers
- Solo founders
- Indie hackers
- Age group: 24–38

Behavior:
- Consumes Instagram Reels and YouTube Shorts regularly
- Interested in productivity and efficiency tools

---

## Objective

To design a reusable prompt engineering system that can generate:

- UGC-style ad scripts
- Scroll-stopping hooks
- Persuasive call-to-action (CTA) lines
- Platform-specific adaptations

---

## Prompt Engineering Approach

The prompts are structured using:

- Defined roles
- Clear objectives
- Input variables
- Output formats
- Tone and style constraints

Each script follows a proven framework:

Hook → Problem → Solution → Transformation → CTA
